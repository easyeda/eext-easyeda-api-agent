# 嘉立创EDA专业版扩展API Agent

基于大模型的扩展API助手，专为嘉立创EDA专业版设计。通过自然语言或语音交互，用户可以便捷地调用EDA扩展API功能，无需记忆复杂的 API 调用方法。

# 主要功能

## AI智能助手
- 支持调用 DeepSeek，提供智能对话体验
- 支持自然语言理解和API调用
- 智能工具调用和结果解析

## 语音交互
- 支持多语言语音识别（中文、英文、日文等）
- 实时语音转文字功能
- 语音状态可视化反馈
- 一键语音输入，提升操作效率

## Markdown支持
- 完整的Markdown格式解析和渲染
- 代码语法高亮显示
- 支持表格、列表、引用等丰富格式
- 优化的聊天消息显示效果

## 灵活配置
- 自定义API地址和密钥
- 可调节的温度参数
- 语音识别语言设置
- 本地设置自动保存

## 调试工具
- 详细的API调用日志记录
- 请求和响应数据查看
- 工具调用状态监控
- 错误信息追踪
