# EasyEDA Pro Extension API Agent

[中文](./README.md)

An extension API assistant based on large language models, specifically designed for EasyEDA Professional Edition. Through natural language or voice interaction, users can conveniently call EDA extension API functions without having to memorize complex API calling methods.

# Main Features

## AI Intelligent Assistant
- Supports DeepSeek integration, providing an intelligent conversation experience
- Natural language understanding and API calling
- Intelligent tool invocation and result parsing

## Voice Interaction
- Supports multilingual voice recognition (Chinese, English, Japanese, etc.)
- Real-time speech-to-text functionality
- Visual feedback for voice status
- One-click voice input, improving operational efficiency

## Markdown Support
- Complete Markdown format parsing and rendering
- Code syntax highlighting
- Support for tables, lists, quotes, and other rich formats
- Optimized chat message display effects

## Flexible Configuration
- Custom API address and key
- Adjustable temperature parameters
- Voice recognition language settings
- Automatic saving of local settings

## Debugging Tools
- Detailed API call logging
- Request and response data viewing
- Tool call status monitoring
- Error information tracking
